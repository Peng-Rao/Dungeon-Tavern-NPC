Detailed Description

Splash / start menu shown over the live scene: Start, Controls, Quit.

Active on boot and whenever the player presses Esc in game. While active the app leaves the cursor free and suspends player input; the tavern keeps simmering behind the menu (flames flicker, the merchant patrols) as a backdrop. The app consumes the start/quit requests raised by the buttons.

Header-only: the only consumer is DungeonApp.cpp.
