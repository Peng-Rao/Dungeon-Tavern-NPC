Detailed Description

The merchant's buy/sell screen.

Gold, merchant stock and the player's inventory live here as plain members, so the state persists for the whole session: close the shop, wander off, come back — your purchases and remaining gold are exactly as you left them. Selling pays half the buy price, rounded down.

The app opens/closes the shop (and manages cursor capture around it); the X button and "Leave shop" raise a close request the app consumes.

Header-only: the only consumer is DungeonApp.cpp.
