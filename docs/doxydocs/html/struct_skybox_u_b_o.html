Detailed Description

Uniforms for the skybox pipeline.

mvpMat is the rotation-only view-projection (the cube tracks the camera so the sky never translates); sunDirDay carries the to-sun direction (xyz) and day factor (w) so the sky can dim to night and place a sun/moon disc; sunColor is the current sun/moon tint.
