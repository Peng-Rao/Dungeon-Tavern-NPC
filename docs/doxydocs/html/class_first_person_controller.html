◆ viewMatrix()

glm::mat4 FirstPersonController::viewMatrix ( ) const
inline

View matrix, built as the inverse of the camera's world placement.

A camera placed in the world would be T(pos) * Ry(yaw) * Rx(pitch); the view matrix is its inverse, which reverses the order and negates each part: View = Rx(-pitch)^-1 ... = Rx(pitch) * Ry(yaw) * T(-pos) (no roll term, since this is a yaw/pitch-only first-person camera). This is the course's "view = inverse of the camera transform" construction. It is mathematically identical to glm::lookAt(pos, pos + forward, {0,1,0}) for the forward vector computed from this yaw/pitch — same matrix, expressed with the angles we already track instead of through a look-at target.
