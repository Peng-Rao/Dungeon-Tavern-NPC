Detailed Description

Continuous, automatic day/night cycle.

Header-only and free of any Vulkan resources (same convention as the other src/ modules). It advances a normalized clock on its own every frame there is deliberately no way for the player to pause or scrub it and derives the sun/moon direction, colour and brightness from that clock. The resulting State drives the directional light in the main shader, the skybox tint, and the sun's shadow map.
