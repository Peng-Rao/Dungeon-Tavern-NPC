⚖ Exam Rules Summary

🗓

Deadlines

  • No hard project deadline, but must complete before last call Feb 2027
  • Register on WeBeep Form when ~40% done
  • Register on Servizi Online for each session
🎭

Presentation

  • All members must be present on the same day
  • 10-minute demo — hard cutoff, no extensions
  • Individual Q&A after demo
🏆

Scoring

  • Project quality: up to 25 points
  • Individual Q&A: up to 8 points
  • Laude possible in exceptional cases
  • Score communicated after code analysis
🚨

Zero-tolerance Rules

  • "I don't remember" = 0 pts for that question
  • Not compiled by professors = fail
  • Evidence of not being done by students = redo project
  • AI tools only for minor contributions
🔁

Retry Rules

  • One retry per session (June/July/Sep, Jan/Feb)
  • Must improve project or re-discuss
  • Members can independently accept or reject marks
  • Accepting members leave the group
💎

Bonus Points

  • Custom 3D assets (own or sourced)
  • Animation, physics, NPC AI
  • Splash screen, menus, scene changes
  • PBR / IBL, normal maps, Toon shader
  • Overall visual quality
